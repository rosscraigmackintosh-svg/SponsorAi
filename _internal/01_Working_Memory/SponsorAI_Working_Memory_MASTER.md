---

2. Non-Negotiable Principles

- No rankings or “best” language

- FanScore = attention only

- FitScore = alignment only

- Budget = access filter only

- Never merge signals into one master score

- Calm, analytical tone

- Trust over engagement

- Intelligence-first positioning

If this section changes, update immediately.

---

3. Core Objects

FanScore

Definition:

Measures digital attention gravity.

Primary inputs:

- Followers

- Engagement

- Growth

Explicitly not:

- Value

- Quality

- ROI

- Performance judgement

---

FitScore

Definition:

Measures brand–property alignment.

Primary inputs:

- Geography

- Category alignment

- Budget overlap

Explicitly not:

- Popularity

- Performance

- Audience size

---

Opportunities

Definition:

Structured sponsorship openings published by properties.

Rules:

- Must include minimum investment

- Filtered by budget + FitScore

- Not ranked by SponsorAI

- Presented as exploration, not recommendation

- Must not overshadow FanScore
