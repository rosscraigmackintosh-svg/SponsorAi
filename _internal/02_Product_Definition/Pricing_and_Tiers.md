## 1. SponsorAI Pricing & TiersStatus: Institutional ModelPositioning: Governance Infrastructure

---

Core Philosophy

Revenue from access.

Never from influence.

No:

- Featured placements

- Boosted properties

- Ranking sales

- Visibility upgrades

---

Tier Structure (Canonical)

1. Professional Access

Intended for:

- Brand managers

- Sponsorship leads

Access:

- Explore

- Compare

- Portfolio

- Controlled exports

- FanScore visibility

- (FitScore future)

No API.

---

2. Enterprise

Intended for:

- Global brands

- Federations

- Procurement-led organisations

Includes:

- All Professional features

- Governance dashboards

- Multi-user controls

- Scenario history tracking

- Version audit logs

- API access (restricted)

- Custom onboarding

- SLA support

Enterprise pricing model:

- Platform fee

- Per-seat licensing

Final pricing bands: Not defined here (uncertainty flagged)

---

API Access Rules

API allows:

- Programmatic access to FanScore data

- Scenario retrieval (private only)

- Export automation

API cannot:

- Reconstruct leaderboard

- Remove confidence

- Strip definitions

- Create global rank ordering

---

Renewal Logic

SponsorAI renews based on:

- Institutional defensibility

- Audit readiness

- Longitudinal portfolio memory

- Governance documentation

Not based on:

- Traffic

- Volume

- Feature hype
