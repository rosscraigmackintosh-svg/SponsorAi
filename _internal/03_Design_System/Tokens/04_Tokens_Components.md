# Component Tokens

These define behaviour at the component level.

## Button

--button-primary-bg: var(--accent)
--button-primary-bg-hover: var(--accent-hover)
--button-primary-text: #FFFFFF

--button-secondary-border: var(--border)
--button-secondary-text: var(--text-primary)

## Card

--card-bg: var(--surface)
--card-border: var(--border)
--card-selected-border: var(--accent)

## Metric Tile

--metric-bg: var(--surface)
--metric-border: var(--border)
--metric-active-border: var(--accent-border)

## Chip

--chip-bg: var(--surface)
--chip-border: var(--border)
--chip-active-border: var(--accent)
--chip-active-text: var(--accent)
